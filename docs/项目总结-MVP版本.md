# WebGIS Agent MVP 版本 - 项目总结

## 项目概述

WebGIS Agent 是一个基于 Cesium.js 和大语言模型的智能地理信息系统，通过自然语言交互实现空间数据分析和可视化。

## 快速启动

```bash
# 1. 安装依赖
cd frontend && npm install
cd ../backend && npm install

# 2. 配置环境变量
cp .env.example .env
# 编辑 .env 文件，填入 API Key

# 3. 启动服务
# 终端 1：启动后端
cd backend && npm run dev

# 终端 2：启动前端
cd frontend && npm run dev

# 4. 访问应用
open http://localhost:3000
```
